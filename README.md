# Training Diary

Uma aplicação Android para registrar e acompanhar seus treinos e progresso físico.

## Descrição

O Training Diary é um diário de treinos que permite aos usuários registrar sessões de exercícios, acompanhar o progresso ao longo do tempo e visualizar estatísticas de desempenho. Ideal para atletas, entusiastas de fitness e qualquer pessoa que queira manter um registro organizado de suas atividades físicas.

## Funcionalidades

- **Registro de Treinos**: Adicione exercícios, séries, repetições e pesos.
- **Acompanhamento de Progresso**: Visualize gráficos e estatísticas de evolução.
- **Interface Intuitiva**: Navegação fácil entre telas de treino, progresso e configurações.
- **Personalização**: Adapte os treinos às suas necessidades.

## Instalação

1. Clone o repositório:
   ```
   git clone https://github.com/seu-usuario/training-diary.git
   ```

2. Abra o projeto no Android Studio.

3. Sincronize o Gradle e execute o app em um dispositivo ou emulador.

## Requisitos

- Android Studio 4.0 ou superior
- SDK do Android 21 ou superior
- Dispositivo Android ou emulador

## Como Usar

1. Na tela principal, selecione "Novo Treino" para começar.
2. Adicione exercícios e detalhes.
3. Salve o treino e visualize o progresso na tela de progresso.
